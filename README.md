
##  Project Description

This project implements a **Singly Linked List** using C++.
A singly linked list is a dynamic data structure where each node contains **data** and a **pointer to the next node**.

The project demonstrates basic linked list operations such as insertion, deletion, counting nodes, and sorting elements.

---

##  Features / Operations

The following operations are implemented in this project:

* Insert node at beginning
* Insert node at end
* Insert node at specific position
* Delete node
* Display linked list
* Count total nodes
* Sort the linked list

##  Technologies Used

* **Programming Language:** C++
**Concepts Used:**
  * Pointers
  * Structures / Classes
  * Dynamic Memory Allocation
  * Linked List Data Structure

## 5️⃣ Structure of Node

Each node in the linked list contains:

* Data field
* Pointer to next node

##  Functions Used in Project

| Function     | Description       |
| ------------ | ----------------- |
| insert()     | Insert new node   |
| deleteNode() | Delete a node     |
| display()    | Show all nodes    |
| count()      | Count total nodes |
| sort()       | Sort linked list  |

This project helps understand the **working of singly linked list** and basic data structure operations.
